# Spherical Camera EIS

## Overview

`Spherical-Camera-EIS` is a project aimed at electronic image stabilization (EIS) for spherical cameras.

> Currently, `src/` and `docs/` are placeholders.

## Purpose

- Research and development of electronic stabilization for spherical camera footage
- Investigation of stabilization algorithms for 360-degree and VR video
- Establish a baseline for future implementations and prototypes

## Directory Structure

- `src/` - source code, algorithm implementation, and experiment code
- `docs/` - project documentation
- `LICENSE` - license information
- `README.md` - this document

## Usage

1. Clone this repository:

```bash
git clone https://github.com/KenjiNM/Spherical-Camera-EIS.git
cd Spherical-Camera-EIS
```

2. Add source code to `src/` and develop your algorithm implementation.

3. Add usage notes, design details, or documentation to `docs/`.
